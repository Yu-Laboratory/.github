# 欢迎来到俞研究室 🎉

## 实验室简介
俞研究室致力于研发面向家庭康复与护理支持的机器人基础技术。实验室设有三个研究小组，成员主要在各自小组内开展研究，同时也会根据需要跨组协作或开展衍生课题。

| 研究小组 | 研究方向 |
|-----------|-----------|
| **生体监测** | 非接触式生命体征测量、排尿预测、居家监护机器人 |
| **电生理仿真** | 电刺激下的疼痛缓解、控制与仿真 |
| **软体机器人** | 手术辅助与康复应用 |
| **其他** | MREPT（磁共振电性质层析成像）等 |
